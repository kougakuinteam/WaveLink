# 後期制作ためのリポジトリ

### 製作者へのお知らせ　:　何があったとしても、リポジトリーにアップグレードしましょう。とりあえず基本的な操作を勉強しましょう。

### 以下は、基本的なgitコードを提示します
#### まずは、キットハープの該当リポジトリのリンクしましょう仕方は：<mark>git remote add origin https://github.com/kougakuinteam/WaveLink.git </mark>
#### 手からに既にコミットされたファイルをこのコードでアップロードします　：<mark>git push origin （ここはpushしたいブランチの名前です,一般的はmain）</mark>
